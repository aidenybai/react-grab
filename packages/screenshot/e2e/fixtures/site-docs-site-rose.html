Quickstart

Capture your first pixel-accurate screenshot in under a minute.

Capture a node

Call captureNode with any element. The returned handle exposes lazy encoders so you only pay for the formats you use. 

// capture and copy to clipboard
import { captureNode } from "fast-html-to-image";

const result = await captureNode(element, { scale: 2 });
const blob = await result.toBlob();
Note: repeat captures of an unchanged DOM reuse the encoded PNG automatically — warm captures typically complete in under 100ms.
