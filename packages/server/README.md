## React Grab Server

Learn more: [docs](https://github.com/aidenybai/react-grab#readme)

The React Grab server is a backend for React Grab.

### Usage

```bash
npx react-grab-server
```

### Configuration

The server can be configured with a `react-grab.config.json` file in the root of the project.

```json
{
  "port": 6243,
  "provider": "claude-code"
}
```
